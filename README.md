# Residence des Muses - Clone Frontend

Vue 3 project with the full frontend content of [residence-des-muses.ch](https://residence-des-muses.ch/).

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Pages

| Page | Route | Content |
|------|-------|---------|
| Projet | `/projet` | Project overview, materials, architecture |
| Situation | `/situation` | Location, context, maps |
| Appartements | `/appartements` | Layouts, energy, budgets, exteriors |
| Galerie | `/galerie` | Gallery with lightbox |
| Plans & Prix | `/plans` | Apartment table, statuses, PDF downloads |
| Telechargements | `/telechargements` | Brochure and plans |
| Contact | `/contact` | Contact form and team details |

## Mobile

- Responsive layout
- Burger menu below `768px`
- Card-based plans section on small screens
- Touch-friendly carousel controls
